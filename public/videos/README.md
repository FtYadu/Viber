# Hero Background Video

This directory should contain the hero background video file named `hero-background.mp4`.

For production, replace this placeholder with an actual video file that:
- Is optimized for web (compressed, ideally < 5MB)
- Has dimensions of at least 1920x1080
- Is in MP4 format with H.264 encoding
- Is 10-15 seconds long and loops seamlessly
- Has subtle motion that doesn't distract from the content

## Video Recommendations
- Subtle abstract patterns or gradients
- Slow-motion professional footage
- Coding/technology themed visuals
- Dark overlay will be applied to ensure text readability

## Fallback
If the video fails to load or takes too long, the component will automatically fall back to a gradient background.
